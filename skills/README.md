# Paper Agent Skills

AI-assisted research workflows for common academic tasks.

Skills are structured prompts that guide AI assistants through complex research workflows using Paper Agent's MCP tools. They help you accomplish multi-step tasks efficiently.

## Available Skills

| Skill | Purpose | Steps |
|-------|---------|-------|
| `literature-review.md` | Generate a comprehensive literature review section | 5 |
| `paper-analysis.md` | Deep analysis of a single paper | 6 |
| `research-gap.md` | Identify research gaps and generate hypotheses | 4 |
| `daily-briefing.md` | Get a daily research briefing from new papers | 3 |
| `writing-assistant.md` | Academic writing with proper citation support | 4 |
| `systematic-review.md` | Conduct a systematic literature review | 7 |

## How to Use

1. Ensure the Paper Agent MCP server is running
2. Open any MCP-compatible AI assistant (Claude, VS Code Copilot, etc.)
3. Load a skill by pasting its content or referencing it
4. Follow the step-by-step workflow

## Creating Skills

Skills follow a simple format:
- **Context**: What the skill does and when to use it
- **Prerequisites**: What MCP tools are needed
- **Workflow**: Step-by-step instructions for the AI
- **Output**: Expected deliverable format
