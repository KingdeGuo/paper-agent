📚 Save to Paper Agent

Drag this button to your bookmarks bar to save papers from any website with one click.

📋 Save to Paper Agent

📖 How to Install

  1. Drag the blue button above to your browser's bookmarks bar
  2. Navigate to any academic paper page (arXiv, PubMed, IEEE, any publisher)
  3. Click the bookmarklet to detect and save the paper

⚙️ Configuration

By default, the bookmarklet connects to http://localhost:8000. To use a different server or add authentication:

  1. Open your browser's developer console (F12)
  2. Run: localStorage.setItem('pa_api', 'https://your-server.com/api')
  3. If using auth: localStorage.setItem('pa_token', 'your-jwt-token')

🔍 What It Captures

The bookmarklet automatically detects:

  • Title (from <meta name="citation_title"> or page title)
  • Authors (from <meta name="citation_author">)
  • DOI (from <meta name="citation_doi">)
  • Publication date
  • Page URL (for arXiv import, PDF download, etc.)

Supported sites: arXiv, PubMed, IEEE Xplore, Springer, ACM, ScienceDirect, Google Scholar, bioRxiv, OpenReview, and any site with standard citation meta tags.
